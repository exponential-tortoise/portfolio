My recent projects

  • Customer Churn prediction :
    A multi layer perceptron classifier based web app to predict customer churn in context to telecom industry.This project has been deployed to heroku utilizing Flask web framework
    Link to live project
    Source code

  • A Lasso regularized Logistic Regression model to detect COVID-19 from Raman spectroscopy dataset :
    This project is regarding a Lasso Regularized Logistic Regression model intended to detect covid-19 from Raman spectroscopy.As the number of features was very large compared to number of observations , we resorted to Lasso regression to avoid overfitting . The model is initialized with a liblinear solver along with L1 (lasso) penalty.The model achieved 97% accuracy on the test dataset .
    Source Code

  • Pulsar Star Prediction by class weight balanced Random forest classifier :
    A random forest classifier model to distinguish pulsar stars from others . The dataset used in this project is HTRU2. The model achieved 97.82 % accuracy on the test dataset .This is an example of class imbalance problem. So, to handle the class imbalance problem , I leveraged the setting of random forest classifier's "class_weight=balanced" wherein classes are automatically weighted inversely proportional to how frequently they appear in the data.
    Source Code

  • Optical Digit Recognition With Deep Learning :
    This project is a multi -class classification of digits from mnist dataset done by Keras framework
    Source code
